DYNAMIC TEST USERS (created per test run by factories)
------------------------------------------------------

These accumulate in the local SQLite database across runs. They are isolated
by unique email/username so they don't collide between runs. To clear them,
reset the database (see README.md — Resetting the database).

NOTES
